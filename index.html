Deploy TailwindNext.js to Netlify

A modern guide to deploying your TailwindNext.js application on Netlify with step-by-step instructions

00

Create Next.js

Create your Latest version of next.js App with the following commands:

npx create-next-app@latest
01

Configure Next.js

Update your next.config.ts file with the following configuration:

const nextConfig = {
  output: 'export',
  images: {
    unoptimized: true,
  },
  reactStrictMode: true,
};
02

Update Package.json

Add the export script to your package.json:

"scripts": {
  "dev": "next dev",
  "build": "next build",
  "export": "next build"
}
03

Build and Export

Run the following command to build and export your Next.js app:

npm run export
04

Create specific Nextjs Version

Run the following command to build and create a specific Next.js version of your app:

npx create-next-app@15.1.6 project-name
